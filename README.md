# webpack-framework7
基于Framework7，使用webpack搭建的前端项目架构


### 安装
```bash
git clone -b dev git@github.com:singcl/webpack-f7.git

cd webpack-f7

npm install 
```

### 开发

```bash
npm run dev
```

### 打包

```bash
npm run build
```
